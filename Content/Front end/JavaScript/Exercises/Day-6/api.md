##JSON
https://www.json.org/json-en.html

JSON is a text data format

```json
45
```
It cannot contains :
- undefined
- Map, Set, Symbol, etc. 
- Classes
- Functions
- Objects like Date

It can contain: 
- null
- primitives ("string", true, 56)
- objects {}
- arrays []

No trailing comma after the last item of a list.
Always double quotes.
